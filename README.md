# PersonalWebsite
## 2018年2月14日
个人网站，主要是移动版，后期会对pc版做兼容，目前开发了
- 首页
- 文章页基本模块
- 组件页按钮和弹窗模块

### 使用
**可以通过**    
npm i 安装模块  
npm start 启动项目，在浏览器中localhost:3060查看，自己改地址。。。   

也可以直接打开dist中的index.html查看
